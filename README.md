Stopwords Collator
=======

Organize raw stopwords into a JSON and a text file.
It does the following:
- Strip CRs
- Strip BOMs
- Strip duplicates
- Strip whitespaces
- Split tabbed words

* Follow the [Hapi coding conventions](http://hapijs.com/styleguide)
* Submit an issue first for significant changes.
